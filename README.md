# oraclejdk-7u80

## Getting Started

Download jdk-7u80-linux-x64.tar.gz from [Oracle](https://www.oracle.com/java/technologies/javase/javase7-archive-downloads.html).

Put the jdk in this directory.

Run:

```sh
docker build -t enakamura/oraclejdk:7u80 .
```
